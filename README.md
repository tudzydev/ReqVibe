# ReqVibe Requirements Analyzer (เครื่องมือวิเคราะห์ข้อกำหนดความต้องการ Interview X)

เครื่องมือนี้คือ Web Application ที่ถูกสร้างขึ้นมาเพื่อสกัด วิเคราะห์ จำแนกกลุ่มความต้องการ (Functional / Non-Functional Requirements) และจัดลำดับความสำคัญของข้อคิดเห็นป้อนกลับจำนวน 102 รายการจากกลุ่มผู้ตอบแบบสอบถามระบบเตรียมสัมภาษณ์งานอัจฉริยะ **Interview X** 

---

## 🚀 คุณสมบัติเด่นของระบบ (Key Features)
1. **แดชบอร์ดหลัก (Analytics Dashboard):** สรุปผลเชิงสถิติ, สัดส่วนการจัดกลุ่มโมดูลความต้องการระบบในรูปแบบพายชาร์ต (Pie Chart) และแถบสถิติความพร้อมโครงการ
2. **ตารางข้อกำหนด (Tabular View):** แสดงรายการทั้งหมดพร้อมฟิลเตอร์คัดกรองแยกกลุ่ม, ระดับความสำคัญ, สถานะ และมีช่องเสิร์ชค้นหาข้อมูลในตาราง ตลอดจนการแก้ไข/ลบรายการโดยตรงผ่าน Modal
3. **บอร์ดจัดกลุ่มความรับผิดชอบ (Kanban Board):** รองรับการลากวาง (Drag and Drop) เพื่อจัดกลุ่มความต้องการระบบเข้าสู่โมดูลการทำงานหรือแก้ไขระดับความต้องการแบบยืดหยุ่น
4. **การวิเคราะห์ระบบและจัด 10 อันดับความสำคัญ (FN/NF & Top 10 Analysis):**
   * **การสกัดประเภทข้อกำหนดอัตโนมัติ:** จำแนก Functional (FR) และ Non-Functional (NFR) ด้วย Heuristics อัจฉริยะ
   * **การคำนวณลำดับความเร่งด่วนสูงสุด 10 ลำดับแรก:** ใช้สูตรถ่วงน้ำหนักความสำคัญและสถานะความคืบหน้าของงานเพื่อเฟ้นหารายการเร่งด่วนที่สุด
   * **ระบบกรองข้อความแบบเรียลไทม์:** ค้นหาและคำนวณ Top 10 ใหม่ตามคำค้นหาได้ทันที
   * **คลิกเพื่อเปิดหน้าต่างแก้ไข:** รองรับการคลิกที่ตัวการ์ดในทุกคอลัมน์เพื่อแก้ไขหรือลบ
5. **การส่งออกรายงาน (Report Exports):** รองรับการดาวน์โหลดรายงานสรุปความต้องการฉบับสมบูรณ์ในรูปแบบ Markdown (`.md`) หรือเอกสารตาราง CSV

## 🛠️ เทคโนโลยีที่ใช้พัฒนา (Tech Stack)
* **HTML5:** โครงสร้างเว็บเซแมนติกและโครงร่างเลย์เอาต์
* **Vanilla CSS3:** ตกแต่งสไตล์แบบพรีเมียมธีมมืด (Premium Dark Mode) ผสมผสานเอฟเฟกต์เบลอหลังกระจก (Glassmorphism) และการจัดเรียง Responsive Layout
* **Javascript (ES6):** ควบคุมการทำงานของระบบทั้งหมด, การจัดการสถานะข้อมูล (State Management), และการสกัดจัดกลุ่ม
* **Chart.js:** แสดงผลกราฟสถิติ
* **Lucide Icons:** ไลบรารีไอคอนสัญลักษณ์

## 💻 วิธีการเปิดใช้งานเครื่องมือวิเคราะห์ (How to Run)

### 1. การเตรียมเครื่องมือ
ตรวจสอบให้แน่ใจว่าติดตั้ง Node.js บนเครื่องของคุณเรียบร้อยแล้ว

### 2. การเริ่มรันเซิร์ฟเวอร์
เปิด Terminal ในโฟลเดอร์โปรเจกต์นี้ จากนั้นรันคำสั่ง:
```bash
npm run dev
```

ระบบจะเปิดใช้งาน Dev Server บนพอร์ตท้องถิ่น เช่น:
* **[http://localhost:5173/](http://localhost:5173/)**

### 3. การโหลดข้อมูลตัวอย่าง
เมื่อเข้าสู่หน้าเว็บครั้งแรก ให้กดปุ่ม **"โหลดข้อมูลตัวอย่าง" (Load Demo Data)** เพื่อดึงชุดคำตอบแบบสอบถามจริงจำนวน 102 รายการเข้ามาวิเคราะห์บนแดชบอร์ดทันที
